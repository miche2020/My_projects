Основной задачей данного проекта были:
1) формирование предположения и гипотезы для дальнейшего построения модели.
2) проверка качество данных и их очиститка если это необходимо.
3) определиться с параметрами модели.
К сожалению в большинстве столбцов присутсвуют пропущенные значения, я старался удалять их по минимуму,потому что чем больше выборка
(кол-во строк), тем модель будет лучше. Сначала я прошел по всем столбцам, удалил строки в которых содержатся гипер аномальные значения, 
которые выглядит не совсем корректно, затем я конвертировал значения yes/no в числовые, для того чтобы было удобней с ними работать,
Если удалять колонки с нан значениями, то останется приблизительно останется около 90 строк, т. е. урезался бы датасет на 70 проц. 
Интересно было подумать как выбрать нужные столбцы для модели, ведь если создавать новые столбцы с экстремальными значениями других колонок,
то можно отобрать большее кол-во колонок, чем я получил, ведь комбинаций достаточно много, чтобы обработать их вручную. Надеюсь я правильно мыслю и верно 
понял свою задачу.)
